NEXORA Social OS – Interactive MVP
==================================

START
-----
1. Datei "NEXORA_Social_OS_MVP.html" herunterladen.
2. Per Doppelklick in Chrome, Edge, Firefox oder Safari öffnen.
3. Es gibt keinen Build-Schritt, kein Backend und keine Datenbank.

Enthaltene Bereiche
-------------------
- Command Center / Dashboard
- AI Studio
- CRM & Kunden-Health
- Projekte & Kampagnen
- Aufgaben-Board mit Drag-and-drop
- Angebote & Sales Pipeline
- Content Studio & Kalender
- Publishing Queue
- Social Inbox
- Social Listening
- Cross-Channel Analytics
- White-Label Report Builder
- Asset Library
- Team, Auslastung & Zeiten
- Automationen
- Integrationen
- Workspace-Einstellungen und Dark Mode

Alle Daten sind hardcodiert und dienen ausschließlich der Produktdemonstration.
